# 📝 Notes App

A modern and responsive Notes App built using HTML, CSS, and JavaScript. The application allows users to create, manage, and store notes directly in their browser using Local Storage, ensuring that notes remain available even after refreshing the page.

## ✨ Features

- Create new notes instantly
- Edit notes anytime
- Delete unwanted notes
- Automatic data saving with Local Storage
- Notes persist even after browser refresh
- Clean and user-friendly interface
- Responsive design for different screen sizes

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage API

## 📚 Key Concepts Implemented

- DOM Manipulation
- Event Handling
- Dynamic Element Creation
- Local Storage
- CRUD Operations (Create, Read, Update, Delete)
- Responsive UI Design
